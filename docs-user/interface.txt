На основном экране программы располагаются следующие элементы:

\subsection{Заголовок окна (располагается сверху)}
Заголовок окна содержит:
\begin{itemize}
\item название программы (<<Компьютерная модель Солнечной системы>>);
\item системные кнопки (их вид может отличаться в зависимости от используемой ОС): кнопка <<свернуть>>, кнопка <<закрыть>>; кнопка максимизации/минимизации (неактивна, посколько окно имеет заданный изначально неизменяемый размер).
\end{itemize}

\subsection{Карта Солнечной системы (располагается слева)}
Здесь отображены основные объекты Солнечной системы вместе с траекториями их движения. Карта включает в себя Солнце, располагающееся по центру экрана, 8 планет Солнечной системы: Меркурий, Венеру, Землю, Марс, Юпитер, Сатурн, Уран и Нептун, а также их основные спутники: для Земли "--- Луну, для Марса "--- Фобос и Деймос, для Юпитера "--- Ио, Европу, Ганимед и Каллисто. Все тела, кроме Солнца, движутся по орбитам, которые изображены: для планет "--- голубым цветом, для спутников "--- сиреневым цветом. Рядом с каждым объектом указано его название, нажатие на изображение объекта скрывает или снова отображает это название. При выполнении моделирования кометы она также отобразится на карте.

\subsubsection{Навигация по карте}

Для приближения/отдаления карты по отношению к той точке, где сейчас находится указатель мыши, необходимо применить колёсико мыши: движение колёсика мыши вперёд увеличивает масштаб изображения, а движение колёсика назад уменьшает его. Для перетаскивания карты необходимо зажать правую кнопку мыши и осуществлять передвижение с помощью неё. Иногда бывает полезно вернуть карту к исходному состоянию, то есть тому, где Солнце находится по центру, а орбиты всех планет имеют масштаб, позволяющий отобразить их все одновременно на экране. Для этого необходимо нажать на клавиатуре клавишу R.

\subsubsection{Поля ввода <<Масса кометы>> и <<Скорость кометы>>}
Данные поля ввода предназначены для указания данных о комете: её массы и скорости соответственно. Для того чтобы начать ввод, необходимо установить курсор в это поле одинарным нажатием левой кнопкой мыши по нему. Последующее нажатие мыши в другой точке экрана сбрасывает установленный курсор. Данные поля позволяют вводить лишь десятичные цифры и символ точки "--- разделителя целой и дробной части числа. Отличный от этих символов ввод будет игнорироваться. Таким образом, введённые числа окажутся корректными десятичными числами (ввод двух точек в числе недопустим; число, начинающееся с точки, будет рассмотрено как число, целая часть которого равна 0; число, заканчивающееся точкой, будет обработано как число, дробная часть которого равна 0).

\subsubsection{Кнопка <<Смоделировать перелёт кометы>>}
Данная кнопка используется для начала моделирования пролёта кометы с указанными исходными данными через Солнечную систему. После нажатия на кнопку левой кнопкой мыши выполняются следующие действия:
\begin{enumerate}
\item Введённые данные проверяются на корректность. Комета не может иметь нулевую начальную скорость или нулевую массу, поэтому в случае равенства значений в одном из полей нулю (если поле вообще не заполнено, оно также будет рассматриваться как поле с нулевым значением) под кнопкой отобразится надпись: <<Ошибка: задано нулевое значение!>>.
\item Если введённые данные корректны, то под кнопкой отобразится надпись: <<Моделирование выполнено>>, а в точке, координаты которой генерируются случайным образом, появится комета.
\item Комета начнёт осуществлять своё движение по Солнечной системе в соответствии с законами Ньютона.
\end{enumerate}

\subsubsection{Кнопки <<Ускорить>> и <<Замедлить>>}
Данные кнопки меняют скорость движения планет и их спутников в Солнечной системе. В силу особенностей системы при изменении скорости также изменится и текущее положение планет, со своего нового положения они продолжат движение с изменённой скоростью: при нажатии на кнопку <<Ускорить>> скорость увеличивается, а при нажатии на кнопку <<Замедлить>> уменьшается.

\subsubsection{Перечень космических объектов с соответствующими им кнопками <<Скрыть>>/<<Отобразить>>}
Здесь приводится список отображаемых объектов, состоящий из Солнца, планет и спутников. Каждый элемент списка состоит из:
\begin{itemize}
\item Названия объекта. При наведении мыши на название объекта отобразится всплывающая краткая справка о нём: в ней указан тип объекта, его название, планета, вокруг которой он вращается (для спутника), и масса.
\item Кнопки <<Скрыть>>/<<Отобразить>>. Если кнопка имеет зелёный цвет и надпись <<Скрыть>>, то при нажатии на неё сама планета и её орбита перестанут отображаться. Если же кнопка имеет красный цвет и надпись <<Отобразить>>, то при нажатии на неё планета и её орбита вновь начнут отображаться на карте Солнечной системы.
\end{itemize}
