# Linux
Prerequisites: GNU C++ compiler
```bash
git clone https://github.com/pusheen1024/solar-system.git
sudo pacman -Sy raylib # use your distro's package manager
cd solar-system
./run.sh
```
# Windows
1) Download the archive SolarSystem.zip from releases
2) Extract the archive into a directory
3) Go to the directory the archive was extracted to
4) Run SolarSystem.exe with a double click on it
